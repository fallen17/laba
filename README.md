Данный проект - программы на C++, принимающие значения для вычисления параметров трёх фигур:
Окружность
Треугольник
Трапеция

Участники проекта: Стуканов В.И. - написание кода для программы, высчитывающей параметры окружности Лухнёв С.А. - написание кода для программы, высчитывающей параметры трапеции Идимешев Р.Ю. - написание кода для программы, высчитывающей параметры треугольника; создание репозитория
